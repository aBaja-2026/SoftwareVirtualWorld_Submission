class ResourceDescription¶

Bases: object

Contains information about the required resource, which can be processed by a ResourcePool.

Also keeps a reference to the requested resource, once the request has been satisfied.

class StaticConfigResourcePool¶

Bases: cmapi.resource.ResourcePool

A ResourcePool specialization.

The managed resources are Application objects. There is a limited amount of available Application objects, each running on one of the AppNode “application hosts” specified via StaticConfigResourcePool.set_app_nodes.

The available AppNode must be set manually, before any resources are requested.

class StaticConfigResourcePoolCarMaker(instance_pool=None, cminstance_range=range(0, 32))¶

Bases: cmapi.resource.StaticConfigResourcePool

CarMaker applications are distributed evenly over the configured AppNode and configured with available CMInstance numbers to avoid collisions.
