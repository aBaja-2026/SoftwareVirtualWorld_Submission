class Application(*args, **kws)¶

Bases: object

Application serves as a base class for applications from the CarMaker Runtime Environment.

Also acts as a frontend for applications in general.

It offers facilities to start and stop processes and pass command line parameters.

To run an arbitrary executable, specify its path with Application.set_executable_path.

Application.set_host in conjunction with Application.set_startup_process_factory_function with an appropriate StartupProcess can be used to start an application on a remote host.

start()¶

Starts the application by spawning a new process with the configured executable.

The exact startup behavior depends on the configured StartupProcess.

Raises:

InvalidStateError – If the application is already running.

Return type:

None

stop()¶

Terminates the application process.

The exact stop behavior depends on the configured StartupProcess.

Raises:

InvalidStateError – If the application is not running

Return type:

None

set_arg(name, *values)¶

Set or add a command line argument for the underlying process.

Parameters:

  • name (str) – Name of the parameter.

  • values (int | str | None) – Optional values following the command line argument name.

Return type:

None

Example

app.set_arg(”–modules”, “sys”, “pathlib”) Multiple arguments with the same ‘name’ must be passed in one call: app.set_arg(”–modules”, “sys”, “–modules”, “pathlib”)
